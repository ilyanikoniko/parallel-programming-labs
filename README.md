## 2. Реализация алгоритма

### 2.1. Основной код (main.cpp)

Разработка велась на языке **C++** с использованием библиотеки **MS-MPI**. Основные элементы реализации:

- **Генерация матриц:** случайные числа с фиксированным seed (`123` для A, `456` для B`) для воспроизводимости
- **Распределение данных:** блочное распределение строк матрицы A между процессами
- **Рассылка матрицы B:** полностью копируется всем процессам через `MPI_Bcast`
- **Вычисления:** умножение с порядком циклов **i-k-j** для улучшения кэш-локальности
- **Сбор результатов:** `MPI_Gather` на процесс 0
- **Замер времени:** `std::chrono::high_resolution_clock` с барьерной синхронизацией

### 2.2. Ключевой фрагмент кода

```cpp
// Очистка файла только при запуске на 1 процессе
    if (rank == 0 && size == 1) {
        ofstream clearFile("data_mpi.txt", ios::trunc);
        clearFile.close();
    }

    for (int n : sizes) {
        if (n % size != 0) continue;

        int rows_per_proc = n / size;
        
        vector<double> A, B(n * n), C;
        vector<double> sub_A(rows_per_proc * n), sub_C(rows_per_proc * n, 0.0);

        // Суммарное время для усреднения
        double total_time = 0.0;

        for (int repeat = 0; repeat < REPEATS; repeat++) {
            if (rank == 0) {
                A.resize(n * n);
                fill_matrix(A, n, 123 + repeat);  
                fill_matrix(B, n, 456 + repeat);
            }

            MPI_Barrier(MPI_COMM_WORLD);
            auto start = chrono::high_resolution_clock::now();

            // Рассылка данных
            MPI_Bcast(B.data(), n * n, MPI_DOUBLE, 0, MPI_COMM_WORLD);
            MPI_Scatter(A.data(), rows_per_proc * n, MPI_DOUBLE,
                        sub_A.data(), rows_per_proc * n, MPI_DOUBLE, 0, MPI_COMM_WORLD);

            // Обнуляем sub_C перед вычислениями
            fill(sub_C.begin(), sub_C.end(), 0.0);

            // Умножение (порядок i-k-j)
            for (int i = 0; i < rows_per_proc; i++) {
                for (int k = 0; k < n; k++) {
                    double temp = sub_A[i * n + k];
                    for (int j = 0; j < n; j++) {
                        sub_C[i * n + j] += temp * B[k * n + j];
                    }
                }
            }

            // Сбор результата на процесс 0
            if (rank == 0) C.resize(n * n);
            MPI_Gather(sub_C.data(), rows_per_proc * n, MPI_DOUBLE,
                       C.data(), rows_per_proc * n, MPI_DOUBLE, 0, MPI_COMM_WORLD);
```

## 4. Команды компиляции и запуска

### 4.1. Компиляция

Использовался компилятор **MinGW GCC 15.2.0** (x86_64, posix, seh):

```bash
g++.exe main.cpp -o mpi_app.exe -m64 -DMSMPI_NO_SAL ^
    -I"." ^
    -I"C:\Program Files (x86)\Microsoft SDKs\MPI\Include" ^
    -L"C:\Program Files (x86)\Microsoft SDKs\MPI\Lib\x64" ^
    -lmsmpi
```

| Флаг | Назначение |
|------|------------|
| `-m64` | 64-битная сборка |
| `-DMSMPI_NO_SAL` | Отключает макросы Microsoft SAL (совместимость с MinGW) |
| `-I` | Пути к заголовочным файлам |
| `-L` | Путь к библиотеке |
| `-lmsmpi` | Линковка с MS-MPI |

### 4.2. Запуск экспериментов

PowerShell-скрипт для последовательного запуска:

```powershell
# Удаляем старый файл с результатами
if (Test-Path "data_mpi.txt") { Remove-Item "data_mpi.txt" }

# Список количества процессов
$cores = @(1, 2, 4, 8)

foreach ($c in $cores) {
    Write-Host "--- Запуск на $c ядрах ---" -ForegroundColor Yellow
    & "C:\Program Files\Microsoft MPI\Bin\mpiexec.exe" -n $c ./mpi_app.exe
}
```

Формат выходного файла data_mpi.txt:
```text
<размер_матрицы> <количество_процессов> <время_в_секундах>
```

**Анализ графика:**
